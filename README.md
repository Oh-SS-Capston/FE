# Oh! SS (Capston)

프론트엔드 전용 (React + Vite) 프로젝트입니다.

## 폴더 구조

- **frontend/** – React + Vite (우주 테마 랜딩, 로그인/회원가입 팝업, 레포 분석 페이지)
  - 레포지토리 정보는 GitHub API를 직접 호출합니다.

## 설치

```bash
cd frontend && npm install
```

## 실행

```bash
npm run dev
```

또는

```bash
cd frontend && npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.
